# delivery

Можно зарегистрироваться или войти в свой кабинет, где в шапке будут отображаться корзина(изначально она пуста), бонусы, полученные за оформление заказа, ссылка на изменение личных данных и ссылка на выход из личного кабинета. В main располагаются категории еды, при выборе категории открывается страница с вариантами блюд этой категории. При клике по блюду откроется подробное описание. Можно добавлять позиции в корзину и удалять их, в корзине отображается итоговая сумма и вопрос с использованием бонусов. При клике на кнопку "Оформить заказ" открывается страница, где указан адрес доставки и сумма с вычетом бонусов(если указана такая функция). После оформления заказа корзина очищается и начисляются бонусы.
В панели администратора можно создать категории и блюда к соответствующей категории.
